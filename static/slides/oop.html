## Concepts - Objects (arguably the most important) - Classes - Imports
### Objects - What is an object? - An object is the base data type in Javascript - Denoted with curly brackets ```js var x = [0,1,2,3,4,5] var y = { hello:0, bye:"no you", others: function () { console.log("function ran") } } console.log(x[0]) console.log(y.hello) console.log(y["bye"]) y.others() ``` - Consists of key value pairs - Can contain any other data type
### Classes - What is an class? - A class is a description of an object - A class has - properties (variables) - methods (functions) - Ball example - Balls diameter (property) - Ball bouncing (method) - Ball rolling (???) - Why are they useful - creates readable code, and follows real life
### Class Code ```js class Ball { diameter = 7 name = "Victor" height = 0 constructor(input) { console.log(input) } up() { this.height = this.height + 1 } } var myBall = new Ball("My ball") console.log("My height is " + myBall.height) myBall.up() console.log("My height is now" + myBall.height) ```
## Imports - You can export things from files, and import them in others - Helpful so you can make things structured and concise ```js // in ball.js export default class Ball { diameter = 7 name = "Victor" height = 0 constructor(input) { console.log(input) } up() { this.height = this.height + 1 } } export var hello = "peekabo" ``` ```js // in script.js import Ball, {hello} from "./ball.js" var ball = new Ball("hi") console.log(hello) ```
