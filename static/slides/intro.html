## DV HackClub
***Start building.***
### *What is HackClub?* - Global non-profit run by highschoolers to teach CS the right way, (unlike APCS \**cough*\*) 💻 - Help people get real life experience making fun projects that foster creativity 🎨 - A great community to just have fun and be yourself 🎉
### *FAQ* - Experience? - Nope! Drop in to any workshop and learn something new! - If you already know the topic, you can stay to present something cool you made, or just tinker - Materials Needed? - Just a laptop! Sometimes you might need to install things, but most of the time a chromebook is okay!
### *What will you learn?* - Basics of programming - Introduction to Web Development 🖼️ - Intro to JS and Python 🐍 - Real life development software - Git CLI + GitHub 🌲 - Visual Studio Code 👁️ - The terminal 🤖 - Data Science - Basic data visualization 📈 - Neural Networks 🧠
